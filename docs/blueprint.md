# **App Name**: Zaid's Portfolio

## Core Features:

- Animated Hero Section: Display the name with typing animation for different roles, and include call-to-action buttons.
- Interactive Experience Timeline: Showcase professional experience in an interactive timeline/card layout with smooth animations and transitions.
- Dynamic Skills Display: Present skills with animated bars and icons to visualize proficiency levels in various technologies.
- Project Showcase: Display key projects with descriptions and technology stacks used. Use AI to summarize the project for users.
- Contact Form with Submission Simulation: A static contact form to capture user information and simulate submission behavior with client-side validation and UI feedback.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to create a modern and vibrant feel, symbolizing technology and innovation.
- Background color: Dark navy (#1A202C) to enhance the contrast with the bright primary color and create a professional, dark-mode aesthetic.
- Accent color: Neon green (#39FF14) to highlight key interactive elements and provide a futuristic touch.
- Body font: 'Inter', a grotesque-style sans-serif known for its modern, neutral, and objective look.
- Headline font: 'Space Grotesk', a proportional sans-serif that gives a techy and scientific feel.
- Use a set of minimalist, line-based icons with subtle electric glow effects to complement the modern theme.
- Implement smooth scroll animations, hover effects on cards, and page fade-in effects to enhance interactivity.
